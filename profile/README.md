## Final Web Applications

After learning Web programming fundamentals (HTML/CSS/JavaScript) without
the use of AI coding agents, students were allowed to use coding agents
as part of an *iterative development* workflow using an `AGENTS.md` file.

Though AI might have been used, they designed all the features and behaviours
and tested the software according to the acceptance criteria they defined.
